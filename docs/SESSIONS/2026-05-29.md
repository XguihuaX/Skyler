## 5 · 主线 4 · NCM 风控回归补丁 ship · 5 端点 isinstance 防御

### 现象
- 实测 NCM 触发风控 · `/cloudsearch/get/web` 返 `{"code":200, "result":"frequent_visit"}`
  · `data["result"]` 是 **str** 不是 dict
- 老代码 `data.get("result") or {}` 短路保留 str · 下游 `result.get("songs")`
  → `AttributeError: 'str' object has no attribute 'get'`

### 修法
5 端点(search × 4 type + playlist_detail)加 isinstance dict check · 非 dict
log warn + 返空 list/dict · 不抛。

### 防回归
本补丁与 Patch A/B 同 commit ship(2026-05-31 feat(netease) commit `06436d8`)。
